1. What Is LocalStorage? / 什么是 LocalStorage？

localStorage is a standard web storage API that allows websites to store data persistently in your browser. Unlike cookies, localStorage data:

  • Is not automatically sent to the server with every HTTP request
  • Has no expiration date — data persists until you or your browser explicitly delete it
  • Is scoped to the origin (domain) — only scripts from koalasave.com can access KoalaSave's localStorage data
  • Has a larger storage capacity (typically 5-10 MB per origin) compared to cookies (4 KB)

localStorage 是一种标准的 Web 存储 API，允许网站在您的浏览器中持久存储数据。与 Cookie 不同，localStorage 数据：

  • 不会随每个 HTTP 请求自动发送到服务器
  • 没有过期时间——数据在您或您的浏览器明确删除之前一直存在
  • 仅限于来源（域名）——只有 koalasave.com 的脚本可以访问 KoalaSave 的 localStorage 数据
  • 存储容量更大（通常每个来源 5-10 MB），而 Cookie 只有 4 KB

4. Managing LocalStorage & Cookies / 管理 LocalStorage 和 Cookie

You have full control over the data stored by KoalaSave:

To clear localStorage (deletes ALL your saved goals):

  • Chrome: Settings → Privacy and Security → Site Settings → See all sites → koalasave.com → Clear data
  • Firefox: Options → Privacy & Security → Cookies and Site Data → Manage Data → Remove koalasave.com
  • Safari: Preferences → Privacy → Manage Website Data → Remove koalasave.com
  • Edge: Settings → Cookies and site permissions → Manage and delete cookies and site data → koalasave.com → Clear

To opt out of Google Analytics:

Install the Google Analytics Opt-out Browser Add-on.

To opt out of personalized ads:

Visit Google Ads Settings or the Network Advertising Initiative opt-out page.

您对 KoalaSave 存储的数据拥有完全控制权：

清除 localStorage（删除所有已保存的目标）：

  • Chrome：设置 → 隐私和安全 → 网站设置 → 查看所有网站 → koalasave.com → 清除数据
  • Firefox：选项 → 隐私与安全 → Cookie 和网站数据 → 管理数据 → 删除 koalasave.com
  • Safari：偏好设置 → 隐私 → 管理网站数据 → 删除 koalasave.com
  • Edge：设置 → Cookie 和网站权限 → 管理和删除 Cookie 及网站数据 → koalasave.com → 清除

选择退出 Google Analytics：

安装 Google Analytics 选择退出浏览器插件。

选择退出个性化广告：

访问 Google 广告设置或 网络广告倡议选择退出页面。
